📖 ภาพรวมระบบ

JST ERP คือระบบบริหารจัดการองค์กรครบวงจร พัฒนาด้วย Laravel 12 ครอบคลุมการบริหารบุคลากร การจัดการคลังสินค้า ระบบยืม-คืนอุปกรณ์ และกระบวนการเบิกจ่ายอุปทาน

URL: http://127.0.0.1:8000 | ฐานข้อมูล: MySQL | PHP: 8.2.12

👥 บทบาทผู้ใช้ (User Roles)

Role คำอธิบาย สิทธิ์การเข้าถึง
admin ผู้ดูแลระบบ เข้าถึงทุกโมดูล
hr ฝ่ายบุคคล จัดการพนักงาน, ลงเวลา
manager ผู้จัดการ ดูรายงานและแดชบอร์ด
inventory ฝ่ายคลังสินค้า จัดการคลังสินค้าทั้งหมด
employee พนักงานทั่วไป ยืม/เบิก ของตัวเอง

🔐 การเข้าสู่ระบบ

Login

POST /login
Content-Type: application/x-www-form-urlencoded

username: your_username
password: your_password

Logout

POST /logout
⚠️ ค่าเริ่มต้น: Username: admin | Password: password — เปลี่ยนทันทีหลังเข้าระบบครั้งแรก!

🌐 API Endpoints - Public (ไม่ต้อง Login)

Method Endpoint คำอธิบาย
GET / Redirect ไปหน้า Login
GET /login แสดงฟอร์ม Login
POST /login ตรวจสอบและเข้าสู่ระบบ

🔑 API Endpoints - Authenticated (ทุก Role)

โปรไฟล์

Method Endpoint คำอธิบาย
GET /profile/edit แก้ไขโปรไฟล์
PUT /profile/update บันทึกการแก้ไข
GET /profile/change-password เปลี่ยนรหัสผ่าน
PUT /profile/update-password บันทึกรหัสผ่านใหม่

การแจ้งเตือน

Method Endpoint คำอธิบาย
GET /notifications ดูการแจ้งเตือนทั้งหมด
POST /notifications/{id}/read อ่านแล้ว
POST /notifications/read-all อ่านทั้งหมดแล้ว

Export Excel

Method Endpoint คำอธิบาย
GET /exports/employees ส่งออกข้อมูลพนักงาน
GET /exports/items ส่งออกข้อมูลสินค้า
GET /exports/borrowings ส่งออกข้อมูลการยืม
GET /exports/requisitions ส่งออกข้อมูลการเบิก

🛡️ API Endpoints - Admin เท่านั้น

Dashboard & ระบบ

Method Endpoint คำอธิบาย
GET /admin/dashboard แดชบอร์ดผู้ดูแลระบบ
GET /admin/activity-logs ดูกิจกรรมในระบบ
GET /admin/health ตรวจสอบสุขภาพระบบ

นำเข้าข้อมูล (Import)

Method Endpoint คำอธิบาย
GET /admin/imports/employees ฟอร์มนำเข้าพนักงาน
POST /admin/imports/employees ประมวลผลนำเข้า (CSV/Excel)
GET /admin/imports/template/employees ดาวน์โหลดเทมเพลต

สำรองข้อมูล (Backup)

Method Endpoint คำอธิบาย
GET /admin/backups รายการ Backup
POST /admin/backups สร้าง Backup ใหม่
GET /admin/backups/{file}/download ดาวน์โหลด Backup
DELETE /admin/backups/{file} ลบ Backup

👔 API Endpoints - HR (admin, hr)

จัดการพนักงาน

Method Endpoint คำอธิบาย
GET /hr/employees รายการพนักงาน (รองรับค้นหา)
GET /hr/employees/create ฟอร์มเพิ่มพนักงาน (รหัส JST-NNN อัตโนมัติ)
POST /hr/employees บันทึกพนักงาน + สร้าง User อัตโนมัติ
GET /hr/employees/{id} รายละเอียดพนักงาน
GET /hr/employees/{id}/edit แก้ไขพนักงาน
PUT /hr/employees/{id} บันทึกการแก้ไข
DELETE /hr/employees/{id} ลบพนักงาน
PATCH /hr/employees/{id}/toggle-block เปิด/ปิดการใช้งาน (active/inactive)

แผนก & ตำแหน่ง

Method Endpoint คำอธิบาย
GET /hr/departments รายการแผนก
POST /hr/departments เพิ่มแผนก
GET /hr/positions รายการตำแหน่ง
POST /hr/positions เพิ่มตำแหน่ง

ลงเวลาทำงาน

Method Endpoint คำอธิบาย
GET /hr/time-records/batch-select เลือกพนักงาน/ช่วงวันที่ เพื่อบันทึกเวลา
POST /hr/time-records/batch-store บันทึกเวลาแบบ Batch (เช้า/บ่าย/OT)
GET /hr/time-records/summary รายงานสรุปรายเดือน
GET /hr/time-records/lock ฟอร์มปิดงวดเวลา
POST /hr/time-records/lock ล็อก/ปลดล็อกงวดเวลา
GET /hr/time-records/logs ประวัติการแก้ไขเวลา (Audit Log)

📦 API Endpoints - Inventory (admin, inventory)

จัดการสินค้า

Method Endpoint คำอธิบาย
GET /inventory/items รายการสินค้า (กรองตามสถานะ)
POST /inventory/items เพิ่มสินค้า (รองรับอัปโหลดรูป)
GET /inventory/items/{id} รายละเอียดสินค้า
PUT /inventory/items/{id} แก้ไขสินค้า
DELETE /inventory/items/{id} ลบสินค้า
PATCH /inventory/items/{id}/toggle-status สลับ available/unavailable

ระบบยืม-คืน

Method Endpoint คำอธิบาย
GET /inventory/borrowing รายการยืมทั้งหมด (กรองตามสถานะ)
POST /inventory/borrowing สร้างใบยืม (อนุมัติอัตโนมัติ, ตัดสต็อก)
GET /inventory/borrowing/{id} รายละเอียดการยืม
PUT /inventory/borrowing/{id} แก้ไขใบยืม
GET /inventory/borrowing/{id}/return ฟอร์มคืนของ
POST /inventory/borrowing/{id}/return ประมวลผลคืน (เพิ่มสต็อกกลับ)

ระบบเบิกของ

Method Endpoint คำอธิบาย
GET /inventory/requisition รายการเบิกทั้งหมด
GET /inventory/requisition/{id}/approve ฟอร์มอนุมัติ
POST /inventory/requisition/{id}/approve อนุมัติ/ปฏิเสธ (ตัดสต็อกเมื่ออนุมัติ)

รายงานคลังสินค้า

Method Endpoint คำอธิบาย
GET /inventory/transactions ประวัติเคลื่อนไหวสต็อก
GET /inventory/transactions/summary สรุปยอดคงเหลือ
GET /inventory/transactions/daily-report รายงานรายวัน
GET /inventory/transactions/category-report รายงานตามหมวดหมู่

👔 ฟีเจอร์ HR

  • ✅ จัดการพนักงานครบวงจร (CRUD)
  • ✅ รหัสพนักงานอัตโนมัติรูปแบบ JST-NNN
  • ✅ สร้าง User Account อัตโนมัติเมื่อเพิ่มพนักงาน
  • ✅ จัดการแผนกและตำแหน่ง
  • ✅ อัปโหลดรูปพนักงาน
  • ✅ ปิดกันการแก้ไขผู้ใช้ที่เป็น Admin
  • ✅ Soft Delete (ซ่อนแทนลบ)

⏰ ฟีเจอร์ลงเวลา

  • ✅ บันทึกเวลาแบบ Batch หลายพนักงาน/หลายวัน
  • ✅ 3 กะ: เช้า (08:00-12:00), บ่าย (13:00-17:00), OT (หลัง 17:00)
  • ✅ สถานะ: มาปกติ, สาย, ลา, ขาด
  • ✅ ล็อกงวดเวลา (ครึ่งเดือนแรก/หลัง)
  • ✅ รายงานสรุปรายเดือน + Export Excel
  • ✅ Audit Log บันทึกการแก้ไขทุกครั้ง

📦 ฟีเจอร์คลังสินค้า

  • ✅ จัดการสินค้าและหมวดหมู่
  • ✅ ติดตามสต็อก (เตือนเมื่อต่ำกว่าขั้นต่ำ)
  • ✅ สร้าง Barcode/QR Code อัตโนมัติ
  • ✅ อัปโหลดรูปสินค้า
  • ✅ ประวัติเคลื่อนไหวสต็อกทุกครั้ง
  • ✅ รายงานคงเหลือ + รายงานรายวัน + หมวดหมู่

🔄 ระบบยืม-คืน

  • ✅ สร้างใบยืมได้เฉพาะสินค้าประเภท equipment/returnable
  • ✅ อนุมัติอัตโนมัติเมื่อพนักงานสร้าง
  • ✅ ตัดสต็อกทันทีเมื่อยืม
  • ✅ คำนวณเกินกำหนดอัตโนมัติ
  • ✅ คืนบางส่วนหรือคืนครบได้
  • ✅ เพิ่มสต็อกกลับเมื่อคืน
  • ✅ แจ้งเตือนของเกินกำหนด

📝 ระบบเบิกของ

  • ✅ เบิกได้เฉพาะสินค้าประเภท consumable
  • ✅ ต้องผ่านการอนุมัติ
  • ✅ ตัดสต็อกเมื่ออนุมัติ
  • ✅ แจ้งเตือนผู้อนุมัติ
  • ✅ ปฏิเสธได้พร้อมเหตุผล

🚀 เริ่มต้นใช้งาน

1. เข้าสู่ระบบครั้งแรก

  1. เปิดเบราว์เซอร์ไปที่ http://127.0.0.1:8000
  2. กรอก Username/Password ของ Admin
  3. กด เข้าสู่ระบบ
  4. เปลี่ยนรหัสผ่านทันทีที่ แก้ไขข้อมูลส่วนตัว → เปลี่ยนรหัสผ่าน

2. เพิ่มพนักงานคนใหม่

  1. ไปที่เมนู HR → จัดการพนักงาน
  2. กด เพิ่มพนักงาน
  3. กรอกข้อมูล (รหัส JST-NNN จะสร้างอัตโนมัติ)
  4. กำหนด Role (hr, inventory, employee)
  5. ตั้ง Username และ Password
  6. กด บันทึก

3. บันทึกเวลาทำงาน

  1. ไปที่เมนู HR → บันทึกเวลาจากบัตร
  2. เลือกแผนกและช่วงวันที่
  3. เลือกพนักงาน
  4. กรอกเวลา เข้า-ออก (เช้า/บ่าย/OT)
  5. เลือกสถานะ (มาปกติ/สาย/ลา/ขาด)
  6. กด บันทึก

4. เพิ่มสินค้าเข้าคลัง

  1. ไปที่เมนู Inventory → รายการสินค้า/อุปกรณ์
  2. กด เพิ่มสินค้า
  3. กรอกข้อมูล (เลือกหมวดหมู่, ประเภท)
  4. กำหนดจำนวนและขั้นต่ำ
  5. อัปโหลดรูป (ถ้ามี)
  6. กด บันทึก

5. สร้างใบยืม

  1. ไปที่เมนู Inventory → สร้างใบยืมใหม่
  2. เลือกพนักงาน
  3. เพิ่มสินค้า (แสดงเฉพาะประเภทที่หยิบคืนได้)
  4. กำหนดวันยืมและวันคืน
  5. กด บันทึก (อนุมัติอัตโนมัติ)

6. นำเข้าข้อมูลจำนวนมาก

  1. ไปที่เมนู Admin → นำเข้าข้อมูล
  2. ดาวน์โหลดเทมเพลต (พนักงาน หรือ สินค้า)
  3. กรอกข้อมูลใน Excel/CSV
  4. อัปโหลดไฟล์ (สูงสุด 5MB)
  5. ดูผลลัพธ์ (สำเร็จ/ล้มเหลว)

7. สำรองข้อมูล

  1. ไปที่เมนู Admin → สำรองข้อมูล
  2. กด สร้าง Backup
  3. รอจนเสร็จ
  4. ดาวน์โหลดหรือลบได้ตามต้องการ
